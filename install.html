How do I get started?

The current stable release (currently 0.1-619) is always available on CRAN and can be installed in R simply by entering: 

install.packages('cheddar')

To use the latest in-development release, download the appropriate file for your system using the blue box at the top of the page and install it in the usual way.

Cheddar's vignettes show round the package:

library(cheddar)
vignette('CheddarQuickstart') # Essential reading
vignette('Community')         # Also essential
vignette('PlotsAndStats')     # Plots and statistics
vignette('Collections')       # Collections of communities
vignette('ImportExport')      # Importing and exporting

Send questions, comments or other feedback to me (lawrence.hudson08 (at) imperial.ac.uk), or use the issue tracker. If you are interested in simulating community dynamics, have a look at the Gruyere package.

Help me get my data into Cheddar!

Researchers typically use their own bespoke data formats. This means that there are probably as many data formats as there are researchers! It is therefore extremely hard to write a generic 'import my data into Cheddar' function.

Help is at hand! If you have community data that you would like to import into Cheddar, please contact me (lawrence.hudson08 (at) imperial.ac.uk) - I will either provide example data-import R code for you to modify or will write the required R code for you.
